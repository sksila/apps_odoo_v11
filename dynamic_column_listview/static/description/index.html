Hide/show the column(s) on the Listview dynamically

This functionality allows user to show/hide column(s) dynamically in the Listview of any object of any installed module. This module will display a button named "Columns" just besides the "Favorites" button in the search options panel, only visible when the List View is active


"Columns" button will display all model field names in a dropdown with a checkbox. To show/hide any field, just check/uncheck that respective checkbox besides field's name in the dropdown and then click on "Save" button.
